【はじめに】
PSPの内部ブラウザのUserAgentを書き換えるソフトです。

-------------------------------------------------------------------------------------
【動作環境】
PSP All Version

-------------------------------------------------------------------------------------
【導入方法】
「UserAgentFaker.prx」と「UserAgentFaker.txt」をms0:/sepluginsにコピー

UserAgentFaker.txtを変えたいUserAgent名にする(最後は改行せず、507文字まで)

「game.txt」、「vsh.txt」に

ms0:/seplugins/UserAgentFaker.prx 1

と追記
